JS盒子模型

JS提供一系列的方法来获取CSS盒子模型中的相关属性

height

假如给元素设置了height，不管内容有多少，获取出来的都是设置的高度；如果没有设置高度，获取出来的就是内容的高度

JS盒子模型属性

总共13种，分为3大类

  • 13个中只有scrollLeft/scrollTop是可以修改的，其他的只能读取不能赋值，赋值无效
  • 使用js盒子模型获取的值都是整数，是浏览器默认四舍五入了

client系列

clientWidth:内容加上左右padding
clientHeight:内容高+上下padding
clientLeft:左边框的宽度
clientTop:上边框的宽度

offset系列

offsetWidth:clientWidth+左右border
offsetHeight:clientHeight+上下border
offsetParent:父级参照物:当前元素的父级参照物，从父级元素开始往上找，找到一个被定位的元素（relative，absolute，fixed），就是当前元素的offsetParent，如果找到body也没有定位元素，那么他的offsetParent就是body
offsetLeft:左偏移量：当前元素外边框距离父级参照物（offsetParent）的内边框的水平距离
offsetTop:上偏移量 ：当前元素的上边框距离父级参照物的内边距的垂直距离
在IE8中左偏移量和上偏移量包含offsetParent的边框，需要判断浏览器的版本

scroll系列

scrollWidth:有溢出的情况下就是内容的宽加上左padding，没有溢出的情况和clientWidth一样
scrollHeight:有溢出的情况下内容的高加上上padding，没有溢出的情况下和clientHeight一样
scrollLeft:内容区域滚出去的宽度
scrollTop:内容区域滚出去的高度
